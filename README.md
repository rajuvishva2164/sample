# 🎂 Dharshini Birthday Project

A private, elegant birthday surprise website for Dharshini.

## 🗂️ Folder Structure

```
dharshini-birthday/
├── index.html          ← Main HTML file (open this in browser)
├── css/
│   └── style.css       ← All styles and animations
├── js/
│   └── app.js          ← Password logic & confetti effect
├── assets/             ← Place any images or fonts here
└── README.md           ← You are here
```

## 🚀 How to Run

1. Open the `dharshini-birthday` folder in **Visual Studio Code**
2. Install the **Live Server** extension (by Ritwick Dey)
3. Right-click `index.html` → **"Open with Live Server"**
4. The site opens in your browser at `http://127.0.0.1:5500`

## ✏️ How to Customize

| What to change         | Where to edit              |
|------------------------|----------------------------|
| Birthday message       | `index.html` → `.message-text` |
| Password               | `js/app.js` → `checkPassword()` |
| Colors / fonts         | `css/style.css` → `:root` variables |
| Add images             | Place in `assets/`, reference in HTML |
